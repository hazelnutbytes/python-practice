#  Python Learning Journey

This repository contains my Python learning exercises and projects, organized by topics. It shows my growth from basic Python concepts to using libraries like NumPy, Pandas, and Matplotlib.  

---

## Topics Covered

- **Lists & Tuples** – operations, slicing, indexing  
- **Sets & Dictionaries** – set operations, key-value handling  
- **Conditionals & Loops** – if-else, for, while, break/continue  
- **Functions** – normal, lambda, *args, decorators  
- **Exception Handling** – try-except, custom exceptions  
- **File Handling** – reading, writing, appending, directories  
- **OOP** – classes, objects, inheritance, polymorphism, encapsulation  
- **NumPy** – arrays, statistics, matrix operations  
- **Pandas** – DataFrames, filtering, grouping, CSV/JSON  
- **Matplotlib** – line, bar, pie charts, customizations
